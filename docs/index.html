Relatable

How to play

Relatable is a game about guessing similar words.

The game starts by giving the computer player four hidden words.
Each turn the computer player generates a set of clues to three of the four words.
Clues are made of new words that each allude to one of the hidden words.
You have to guess which of the hidden words the clues are hinting at.
After you submit a guess the real answers are revealed, which you can use to help in future guesses.

You win when you correctly guess all clues in a turn, on two seperate turns. You start with two turns already revealed.

Good luck!

Mathsy details Relatable works by using word vector embeddings.

Embeddings assign each word in a collection a position in space. They do so with the condition that "similar" words should be mapped "close" positions.

For example if we wanted to create an embedding that mapped furniture to positions on the globe, we might put "chair" and "stool" in Europe so they are close. Then "desk" and "table" in America and "hammock" away in Australia.
Computers can construct much more complicated spaces. They can arrange dictionaries of words in spaces with hundreds of dimensions.
Using such an embedding the AI player of Relatable can find words that are similar to the hidden words and set clues.

Currently clues are built from FastText embeddings. Any distasteful (yeah there are some) clues come from using this data unfiltered.
